# CMM scenario figures (regenerated)

Generated by driving the real Qt workbench under the offscreen platform, fully self-contained
inside `CMM/` (no parent-folder paths). Run from the `CMM/` directory with the project venv:

```bash
QT_QPA_PLATFORM=offscreen CMM_OUTPUT_DIR=./temp_figures_new \
    PYTHONPATH=src .venv/bin/python -m cmm.app.screenshots
QT_QPA_PLATFORM=offscreen CMM_OUTPUT_DIR=./temp_figures_new \
    PYTHONPATH=src .venv/bin/python -m cmm.app.succinate_scenario
QT_QPA_PLATFORM=offscreen CMM_OUTPUT_DIR=./temp_figures_new \
    PYTHONPATH=src .venv/bin/python -m cmm.app.genome_scale_scenario
```

Active solver in this run: **gurobi** (LP + QP + MILP + MIQP), so MOMA/ROOM, rMTA, and the
original-MTA MIQP run for real (not the solver-capability fallback).
